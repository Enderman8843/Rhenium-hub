# Rhenium HUB
It is Hi-Speed USB HUB with USB2514B_B Chip and 24mhz crystal , With Support of USB C aswell as USB A


## How To Use ?
JUST PlUG IT INTO YOUR COMPUTER VIA THE INPUT
